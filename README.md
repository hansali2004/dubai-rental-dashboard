Dubai Rental Analysis Dashboard
An interactive Power BI dashboard analyzing rental market trends across Dubai.

Overview:
This dashboard explores rental pricing patterns across Dubai, helping identify high-demand areas and key market trends through interactive visuals.

Tools Used:
Power BI Desktop — dashboard design and visuals
Power Query — data cleaning and transformation

Features:
Visuals for average rent, pricing trends, and area comparisons
Filters for location, date, property type, and budget and the number of bedrooms
Insights on high-demand areas and rental patterns to support data-driven decisions
